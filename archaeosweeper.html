Install the game

Play offline — add to your home screen

Archaeosweeper is a PWA — it installs like a native app and works fully offline after the first visit. No app store required.

iPhone & iPad
  1. Open this page in Safari
  2. Tap the Share button □↑ at the bottom
  3. Scroll down and tap "Add to Home Screen"
  4. Tap Add — done

Launches fullscreen, works offline.

Android
  1. Open this page in Chrome
  2. Tap the menu ⋮ in the top right
  3. Tap "Add to Home Screen"
  4. Tap Add — done

Chrome may also show an install banner automatically.

Desktop
  1. Open this page in Chrome or Edge
  2. Look for the install icon ⊕ in the address bar
  3. Click it and confirm
  4. The game opens in its own window

Works offline after first load.
